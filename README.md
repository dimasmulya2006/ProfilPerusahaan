PT.MulyaTheScant adalah website profil perusahaan (company profile) di bidang teknologi yang dirancang untuk menampilkan informasi perusahaan, layanan unggulan, serta testimoni klien. Dibangun dengan antarmuka **Glassmorphism** yang elegan, aplikasi ini memastikan pengalaman pengunjung yang modern dan responsif. Di sisi visual, website ini menggunakan perpaduan warna *Navy Blue* dan *Neon Cyan* pada latar belakang terang (*Light Mode*) untuk menciptakan kesan profesional dan futuristik.

## ✨ Fitur Utama

### 🧑‍💻 Fitur Pengunjung (User)
* **Navigasi Interaktif & Smooth Scroll:** Beralih antar bagian halaman (Beranda, Tentang Kami, Layanan, Testimoni) secara halus dengan deteksi bagian aktif otomatis di menu navigasi.
* **Tampilan Glassmorphism Dinamis:** Desain kartu bergaya kaca transparan dengan latar belakang *blob* animasi yang bergerak untuk pengalaman visual yang hidup.
* **Statistik Animasi (Counter):** Angka pencapaian perusahaan (Kepuasan Klien, Proyek Sukses) yang menghitung secara otomatis saat pengguna menggulir layar.
* **Integrasi Kontak & WhatsApp:** Formulir kontak interaktif dan tombol mengambang (Floating Button) WhatsApp untuk memudahkan pengunjung menghubungi tim ahli.

---

## 🛠️ Teknologi yang Digunakan

* **Frontend:** HTML5 Semantic
* **Styling (Desain):** Vanilla CSS3 (Custom Variables, CSS Grid, Flexbox, Keyframe Animations)
* **Interaktivitas:** Vanilla JavaScript (ES6, Intersection Observer API, DOM Manipulation)
* **Desain UI/UX:** Tema warna *Navy Blue* & *Neon Cyan*, efek *Glassmorphism*.

## 💻 Cara Instalasi & Menjalankan (Local Development)

Website profil perusahaan ini bersifat statis (HTML/CSS/JS) sehingga sangat mudah dijalankan tanpa perlu pengaturan server (backend) khusus.

### 1. Menjalankan Aplikasi
1. Clone atau unduh repositori ini ke dalam komputer Anda.
2. Buka folder proyek `PT.MulyaTheScant`.
3. Cukup klik ganda (double-click) pada file `index.html` untuk membukanya di browser (Chrome, Edge, Firefox, Safari).

### 2. Pengembangan Lanjutan (Opsional)
Jika Anda menggunakan Visual Studio Code, sangat disarankan menggunakan ekstensi **Live Server**:
1. Buka folder proyek di Visual Studio Code.
2. Klik kanan pada file `index.html` dan pilih **Open with Live Server**.
3. Browser akan otomatis terbuka dan memuat ulang halaman setiap kali ada perubahan pada kode.
